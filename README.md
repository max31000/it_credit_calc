# Ипотечный стратег

Интерактивный калькулятор сравнения стратегий ипотечного кредитования по льготной ИТ-ипотеке в России.

## Что делает приложение

Помогает заёмщику по льготной ИТ-ипотеке сравнить две финансовые стратегии:
- **Стратегия А**: направлять все свободные деньги в досрочное погашение с уменьшением ежемесячного платежа
- **Стратегия Б**: платить минимальный аннуитет и инвестировать остаток под рыночную доходность

Ключевая функция — **анализ сценария слёта**: что произойдёт с платежом если льготная ставка будет утрачена в различные моменты времени. Калькулятор показывает **точку финансовой безопасности** — момент, начиная с которого даже при слёте с ИТ-ипотеки платёж не превысит исходный льготный уровень.

## Описание графиков

**График 1: Чистое состояние** — показывает динамику netWorth = накопления − долг для обеих стратегий по месяцам горизонта.

**График 2: Накопления и долг** — детальный вид: накопления стратегии Б, остаток долга обеих стратегий.

**График 3: Анализ платежа при слёте** — ключевой график. Ось X = "в какой месяц мог бы произойти слёт". Показывает как изменяется минимальный платёж при слёте в зависимости от момента слёта. Зелёная линия = безопасный порог (текущий льготный платёж).

## Финансовые алгоритмы

Подробная математическая спецификация — в `ALGORITHMS.md`.

- **Аннуитет**: PMT = P × r / (1 - (1+r)^(-n))
- **Стратегия А**: каждый месяц пересчитывается новый аннуитет от уменьшенного остатка
- **Стратегия Б**: S(t) = S(t-1) × (1 + r_dep/12) + invest_monthly
- **Рыночная ставка при слёте**: keyRate − bankDiscount + 1.5%
- **Налоговые вычеты**: имущественный (лимит 2 млн) + по процентам (лимит 3 млн)

При пуше в ветку `master` GitHub Actions автоматически:
